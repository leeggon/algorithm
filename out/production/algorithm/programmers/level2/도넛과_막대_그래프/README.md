## 유형

구현

## 풀이법

각 모양의 그래프마다 특징 점을 잡는 것이 중요했음.

막대 그래프 : 들1 나X 정점이 무조건 존재함.

8자 그래프 : 들2 나2 정점 무조건 존재함.

새로 만든 정점 : 들X 나≥2 (**나 : 총 그래프 개수**)

도넛 그래프 : 도넛 그래프는 모든 정점이 똑같이 생겨서 판단 불가능.

그래서 **나 : 총 그래프 개수** 를 구해서 도넛: 총 그래프 - 막대 - 8자로 구했음.
