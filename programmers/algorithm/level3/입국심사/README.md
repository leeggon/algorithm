## 유형

이분탐색

## 풀이법

가장 오래 소요되는 시간 기준을 right로 설정하고 이분 탐색을 수행한다.

10^18이라는 엄청난 크기의 수이지만 log2(10^18) = 60으로 시간복잡도가 60 * 10^5가 된다.

특정 mid 시간을 지목했을 때, n명보다 많이 수행할 수 있으면 result를 계속 업데이트한다.

(특정 시간을 찾는 것이 아니라 최소 소요시간을 찾는 것이므로)
