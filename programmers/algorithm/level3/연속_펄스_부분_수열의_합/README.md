## 풀이법

펄스파 종류가 2가지 있는데, 각각을 곱한 수열을 우선 구한다.

해당 수열에서, 연속 부분 수열 중 합이 최대인 구간의 합을 구해야 하는데,

각 노드를 방문할 때마다, (이전까지의 합 + 자기자신)과 (자기자신) 중 더 큰 수를 취한다. 그리고 해당 수를 result에 매번 업데이트쳐서 최대 구간 합을 구할 수 있도록 풀이했다.
