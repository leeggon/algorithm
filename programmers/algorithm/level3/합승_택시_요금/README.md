## 유형

플로이드 워샬

## 풀이법

플로이드 워샬 알고리즘을 통해 각 지점에서 다른 지점들까지의 최소 경로를 구해주었다.

시간복잡도 : O(N^3) -> N이 200 이하이기에 가능했다.

만약 합승을 하게 된다면, 경유지까지는 비용을 공유하게 되는 구조이므로,

합승을 안하는 비용 : cost[s][a] + cost[s][b]과

각 경유지에서 합승을 할 때의 비용을 각각 비교해서 계산해주었다.
