## 풀이법

쉽지 않은 DP 문제였다.

n = 8, money = [1,2,5]인 상황에서

처음엔 단순히 dp[n] = dp[n-1] + dp[n-2] + dp[n-5] 의 방식인줄 알았으나, 겹치는 경우의 수들이 존재했다.

풀이법은 DP가 맞는데

1. 1원만 써서 조합을 짜는 경우
2. 1원 + 2원까지 써서 조합을 짜는 경우
3. 1원 + 2원 + 5원까지 써서 조합을 짜는 경우

이런 식으로 행을 구성하고, 열은 내가 조합하고자 하는 수를 뒀다.

이렇게 설계하는 이유는 새로운 동전을 쓰지 않는 경우라면(예를 들어 3번 조합을 구할 때), 이전에 구했던 조합(2번 조합)과 같은 경우의 수가 나오기 때문이다.

새로운 동전을 쓰게 된다면, 그만큼 무게를 뺀 상태에서 그 동전까지 고려한 경우의 수와 같게 구할 수 있다.

dp[i][j] = dp[i-1][j] + dp[i][j - money[i-1]];

따라서, 점화식이 이런 식으로 구성된다.
