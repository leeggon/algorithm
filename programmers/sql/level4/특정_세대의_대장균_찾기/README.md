## 풀이법

MySQL의 WITH문을 처음으로 알게 되었다.

MySQL 8.0 이상부터 지원한다고 하는데, 임시 테이블을 만들지 않고 서브쿼리를 더 가독성 있게 작성하는 방법이라고 한다.

사용했을 때 기대효과는 다음과 같다고 한다.

1. 가독성 향상
2. 재사용 가능
3. 성능 최적화

1세대, 2세대에 해당하는 대장균의 테이블을 WITH로 만들어놓고, 간편하게 3세대 대장균을 구했다.
